# 70. 爬楼梯

### 简述

- [Leetcode](https://leetcode-cn.com/problems/climbing-stairs/)

### 思路

- 动态规划，斐波那契数列

### 代码

`Swift`

```swift
func climbStairs(_ n: Int) -> Int {
    var dp = Array(repeating: 0, count: n+1)
    dp[0] = 0
    if n == 0 {
        return dp.last!
    }
    dp[1] = 1
    if n == 1 {
        return dp.last!
    }
    dp[2] = 2
    if n == 2 {
        return dp.last!
    }
    
    for index in 3...n {
        dp[index] = dp[index-1]+dp[index-2]
    }
    return dp[n]
}

```
