# 575. 分糖果

### 简述

- [Leetcode](https://leetcode-cn.com/problems/distribute-candies/)

### 思路

- 种类总数大于所得平均数，则取平均数，否则取种类总数

### 代码


`Swift`

```swift
func distributeCandies(_ candies: [Int]) -> Int {
    let total = candies.count
    let classes = Set(candies).count
    let average = total / 2
    if classes >= average {
        return average
    } else {
        return classes
    }
}

```
