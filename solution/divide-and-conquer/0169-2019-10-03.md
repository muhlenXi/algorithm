# 169. 多数元素

### 简述

- [Leetcode](https://leetcode-cn.com/problems/majority-element/`)

### 思路

- 1、统计出现次数
- 2、排序后返回中间 index 元素

### 代码


```swift
func majorityElement(_ nums: [Int]) -> Int {
    var dict = [Int: Int]()
    for number in nums {
        if let _ = dict[number] {
            dict[number]! += 1
        } else {
            dict[number] = 1
        }
    }
    
    for (key,value) in dict {
        if value >  Int(nums.count/2) {
            return key
        }
    }
    return 0
}
```
