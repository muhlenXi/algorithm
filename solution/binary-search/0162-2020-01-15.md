# 162. 寻找峰值

### 简述

- [Leetcode](https://leetcode-cn.com/problems/find-peak-element/)

### 思路

- 二分查找

### 代码

`Swift`

```swift
func findPeakElement(_ nums: [Int]) -> Int {
    var low = 0
    var high = nums.count-1
    while low < high {
        let mid = (low + high)/2
        if nums[mid] < nums[mid+1] {
            low = mid + 1
        } else {
            high = mid
        }
    }
    return low
}

```
