# 278. 第一个错误的版本

### 简述

- [Leetcode](https://leetcode-cn.com/problems/first-bad-version/)

### 思路

- 二分查找

### 代码

`Java`

```java
public int firstBadVersion(int n) {
    int low = 0;
    int high = n;
    int mid = low + (high-low)/2;

    while (low < high) {
        mid = low + (high-low)/2;
        if (isBadVersion(mid)) {
            high = mid;
        } else {
            low = mid + 1;
        }
    }
    return low;
}
```
