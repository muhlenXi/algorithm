# 69、x 的平方根

### 简述

- [Leetcode](https://leetcode-cn.com/problems/sqrtx/)

### 思路

- 二分查找，注意循环条件

### 代码
 
 ```swift
func mySqrt(_ x: Int) -> Int {
    if x == 0 {
        return 0
    }
    
    var low = 1
    var high = x/2 + 1
    while high - low > 1 {
        let mid = low + (high-low)/2
        if mid * mid > x {
            high = mid
        } else {
            low = mid
        }
    }
    return low
}
 ```
