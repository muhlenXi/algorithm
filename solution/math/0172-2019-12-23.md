# 172. 阶乘后的零


### 简述

- [Leetcode](https://leetcode-cn.com/problems/factorial-trailing-zeroes/)

### 思路

- 统计 5 出现的次数，每出现一个 5， 就产生一个 0.

### 代码

`Swift`

```swift
func trailingZeroes(_ n: Int) -> Int {
    guard n != 0 else { return 0 }
    guard n > 4 else { return 0 }
    
    var count = 0
    var val = n
    
    while val >= 5 {
        count += val / 5
        val /= 5
    }
    
    return count
}
```
