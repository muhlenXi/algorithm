# 231. 2的幂

### 简述

- [Leetcode](https://leetcode-cn.com/problems/power-of-two/)

### 思路

- 循环除2判断

### 代码

`Swift`

```
func isPowerOfTwo(_ n: Int) -> Bool {
    var nn = n
    guard nn > 0 else { return false }
    guard nn != 1 else { return true }
    
    while nn % 2 == 0 {
        nn = nn / 2
    }
    return nn == 1
}
```
